# Product Overview

This is a TypeScript-first 2D web game engine with an Entity-Component-System (ECS) architecture. The engine is designed for browser-based games with a focus on composability, determinism, and performance.

## Core Features

- ECS-driven architecture with data-oriented design
- WebGL2 rendering with Canvas 2D fallback
- Comprehensive subsystems: rendering, audio, input, animation, physics, assets, scenes, particles, UI, save/load
- Plugin system for extensibility
- Fixed timestep game loop with interpolation

## Showcase Games

The monorepo includes three showcase titles demonstrating engine capabilities:

- Super Snake (implemented)
- Advanced Tetris (placeholder)
- Flappy-like (placeholder)

## Target Audience

Game developers building 2D web games who need a lightweight, modular engine with strong TypeScript support and modern web standards.
