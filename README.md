# Web Game Engine Monorepo

This monorepo hosts the TypeScript-based 2D web game engine and showcase games (Super Snake, Advanced Tetris, Flappy-like), along with documentation, examples, and tests.

## Packages

- `@web-game-engine/core` — engine source located in `packages/engine`.
- `@web-game-engine/super-snake`, `@web-game-engine/tetris-advanced`, `@web-game-engine/flappy-like` — showcase games located in `packages/games/*`.

See the `requirements/` directory for detailed functional and non-functional requirements, roadmap, and acceptance criteria.
